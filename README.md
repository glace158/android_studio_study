This game in which two players play rock-paper-scissors.<br>
### How to play?
1. press the Game start button, you will randomly play one of three things: scissors, rock, and paper.<br>
2. Press the "Who is winning?" button to determine the winner and loser.<br>
3. If the GameCount is greater than or equal to 10, the winning rate is displayed.
### Secret function
If GameCount is over 15..<br>
Player1 is manipulated to a figure whose winning rate is set in the code.<br>
If you set the winning rate to 60%, Player 1 win from 15 over rounds until reaches 60%.
